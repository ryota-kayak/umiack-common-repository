# Changelog

All notable changes to the "Cherry Blossom Kayak" project will be documented in this file.

## [1.0.0] - 2026-04-12

### Added
- Created `README.md` and `CHANGELOG.md` in English for professional documentation.
- Integrated Git and established GitHub repository for cloud backup and version control.
- Designed a unified conversion button system (LINE, Web, and Phone).
- Implemented Phone Consultation button as a dual-line styled button with icon integration.

### Changed
- Refactored legacy HTML into semantic structures (converted `div` containers to `dl/dt/dd` lists for FAQs and Details).
- Optimized gallery styling: forced 4:3 aspect ratio, 12px border-radius, and added hover lift animation.
- Improved typography with `Inter` and `Hiragino` font stack and optimized character spacing.
- Fixed layout regression where FAQ items were not stacking vertically on mobile.

### Fixed
- Resolved file naming conflicts (consolidated `本文 2.html` into `本文.html`).
- Cleaned up redundant template files (`YouTube埋め込み.html`).
